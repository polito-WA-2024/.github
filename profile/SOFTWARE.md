# Web Applications (2023/2024)

This section lists the software applications and the libraries used in the course "_Web Applications_" (in English) for the [Master Degree in Cybersecurity](https://www.polito.it/en/education/master-s-degree-programmes/cybersecurity) at Politecnico di Torino, Italy. Lectures and Labs will be given either by Enrico Masala (EM) or Antonio Servetti (AS).
